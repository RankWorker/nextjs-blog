@rankworker/nextjs-blog

A complete, customizable, SEO-optimized blog layer for Next.js. Add articles, pagination, tags, metadata, structured data, and sitemaps without rebuilding the same plumbing in every project.

$ npm install @rankworker/nextjs-blog
Read the documentation View package

Included

  • Paginated blog index
  • MDX article rendering
  • Paginated tag pages
  • SEO metadata and JSON-LD
  • Blog and image sitemaps
  • Replaceable UI components

Two content modes

01 / LOCAL MDX

Standalone and free

Keep RankWorker-format MDX files in your repository. No account or hosted service required.

02 / RANKWORKER

Headless CMS

Use a RankWorker subscription for Direct API publishing and webhook-driven cache updates.
