# Changelog

All notable changes to `@rankworker/nextjs-blog` will be documented here. Releases
follow Semantic Versioning and are assembled from the Changesets committed with
user-visible changes.

## 1.0.0

- Initial public release.
- Next.js App Router blog facade with local MDX and RankWorker Direct API sources.
- Customizable presentation components, metadata, JSON-LD, sitemaps, webhook
  invalidation, and an initializer CLI.
